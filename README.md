# 🎬 AI Movie Studio

Generate AI-powered short films from a single idea.

## Features

- Story Generator
- Screenplay Generator
- Storyboard
- Character Profiles
- AI Images
- AI Video
- Voice Generation
- Movie Export

## Tech Stack

- Python
- NiceGUI
- Ollama
- MoviePy

## Run

'''bash
pip install -r requirements.txt
python app.py
'''
